Project Description: 
This is a bash script for a book database where the user can add, list, and delete
books. It stores information about a books title, author, page count, year of pulication
and publisher. Titles can be searched by string matching, books from different 
eras listed, and Authors publications can be also searched, as well as listing 
books below a certain page count.

Installation Instructions:
To use the program simply unzip it to the working directory. There should be a
bookRecord.sh and book_database.txt file along with a backup which can be copied
to the book_database.txt if the database is cleared in the program. It is 
book_database.bkp.txt.

Give the shell script permissions using: `chmod 755 bookRecord.sh` and then 
run the program using:

`./bookRecord.sh` 

Menu options are chosen using integers and validation provides prompts for incorrect
entries.
